About me


Saya adalah seorang Web Developer dengan fokus pada pengembangan antarmuka web yang modern, responsif, dan mudah digunakan. Saya memiliki pengalaman dalam membangun website menggunakan HTML, CSS, dan JavaScript, serta mengembangkan antarmuka aplikasi menggunakan React.js.


Saya terbiasa bekerja dengan praktik pengembangan yang baik, seperti penulisan kode yang terstruktur, proses debugging yang sistematis, serta penggunaan Git dan GitHub untuk kolaborasi dan manajemen versi. Saya memiliki kemampuan analisis yang baik, bertanggung jawab terhadap pekerjaan, dan mampu beradaptasi dengan cepat terhadap teknologi serta kebutuhan proyek.


Saat ini, saya terus meningkatkan kompetensi teknis melalui pengembangan proyek dan pembelajaran berkelanjutan, dengan tujuan memberikan kontribusi maksimal dalam pengembangan produk digital yang berkualitas dan berkelanjutan.
